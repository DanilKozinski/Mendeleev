
# Interactive Periodic Table

## Description
This project is an interactive Mendeleev Periodic Table application with chemical reaction balancing capabilities.

## Controls
- **Mouse**: Click on elements to view information
- **Keyboard**:
  - **W, A, S, D**: Navigate between elements
  - **E / Enter**: View selected element details

## Features
- **Element Information**: Detailed data for each chemical element
- **Reaction Calculator**: 
  - Balance coefficients for double displacement reactions
  - Enter reaction → Get balanced equation instantly

## Tech Stack
- **Database**: SQL
- **GUI Framework**: PyQt 5
- **Programming Language**: Python

# Интерактивная Периодическая Таблица

Интерактивное приложение для изучения химических элементов и балансировки уравнений реакций.

## Управление
- **Мышь**: Клик на элемент → информация
- **Клавиатура**:
  - **WASD** — навигация
  - **E/Enter** — просмотр информации

## Функции
- **Просмотр данных** об элементах
- **Калькулятор реакций**:
  - Балансировка уравнений обмена
  - Автоматический расчёт коэффициентов

## Технологии
- **База данных**: SQL
- **Интерфейс**: PyQt 5
- **Язык**: Python
